# skill/ — 技能系统

> 更新时间：2026-05-30

技能模块负责扫描工作区技能目录、解析 `SKILL.md` frontmatter，并为提示词和工具提供技能索引。路径处理逻辑集中在工具函数中，避免不同调用方重复实现目录解析。

## 文件职责

| 文件 | 职责 |
|------|------|
| `__init__.py` | 标记 `skill` 为 Python 包。 |
| `loader.py` | 扫描技能目录，解析技能元数据并按名称构建索引。 |
| `skill_utils.py` | 规范化技能名、解析技能路径并限制路径越界。 |

## 注意事项

- README 只描述当前文件；不要恢复已经不存在的技能策展文件或内置技能目录条目。
