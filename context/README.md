# context/ — 上下文压缩

> 更新时间：2026-05-30

上下文模块负责在长对话接近模型窗口上限时压缩历史内容，尽量保留任务连续性和关键证据。当前实现聚焦工具结果裁剪、结构化摘要、头尾保护和压缩状态恢复。

## 文件职责

| 文件 | 职责 |
|------|------|
| `__init__.py` | 标记 `context` 为 Python 包。 |
| `compressor.py` | 根据 token 预算压缩消息列表，裁剪大型工具结果并生成结构化摘要。 |

## 注意事项

- 压缩策略调整后，需要重点运行 `tests/test_compressor.py` 和涉及 Agent 主循环的测试。
