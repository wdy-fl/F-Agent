# config/ — 配置管理

> 更新时间：2026-05-30

配置模块负责从默认值和 YAML 文件加载运行配置，并集中管理工作区、数据库、日志、记忆和技能等路径。其他模块应通过配置对象读取运行参数，避免分散硬编码。

## 文件职责

| 文件 | 职责 |
|------|------|
| `__init__.py` | 标记 `config` 为 Python 包。 |
| `settings.py` | 定义默认配置、加载 `workspace/config.yaml` 并派生运行路径。 |

## 注意事项

- 新增配置项时需要同时更新 `config.yaml.example`、顶层 `README.md` 配置说明和相关测试。
