# db/ — 会话持久化

> 更新时间：2026-05-30

数据库模块负责 SQLite 持久化，包括 schema 初始化、轻量迁移、会话消息记录、工具调用记录、统计信息和 FTS5 历史搜索。它为 Agent 恢复上下文和检索历史消息提供基础存储能力。

## 文件职责

| 文件 | 职责 |
|------|------|
| `__init__.py` | 标记 `db` 为 Python 包。 |
| `schema.py` | 初始化数据库表、索引、FTS5 虚表和增量迁移。 |
| `session.py` | 提供会话、消息、工具调用、统计和全文搜索的读写接口。 |

## 注意事项

- schema 变更必须保持已有工作区数据库可迁移，并补充对应测试。
