# llm/ — LLM 客户端

> 更新时间：2026-05-30

LLM 模块封装 OpenAI 兼容接口，向 Agent 主循环提供统一的模型调用、流式响应和 token 估算能力。通过 `base_url` 和模型配置支持切换不同兼容服务。

## 文件职责

| 文件 | 职责 |
|------|------|
| `__init__.py` | 标记 `llm` 为 Python 包。 |
| `client.py` | 创建 OpenAI 兼容客户端，执行聊天补全、处理工具调用响应并估算 token。 |

## 注意事项

- 修改流式响应或工具调用解析时，需要同步检查 `agent/loop.py` 的消费逻辑。
