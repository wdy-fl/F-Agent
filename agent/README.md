# agent/ — Agent 核心层

> 更新时间：2026-05-30

Agent 核心层负责把用户输入、系统提示词、LLM 调用、工具执行和会话消息串成 ReAct 循环。这里的代码是 CLI 启动后每轮任务执行的主控制面。

## 文件职责

| 文件 | 职责 |
|------|------|
| `__init__.py` | 标记 `agent` 为 Python 包。 |
| `budget.py` | 管理单轮任务的迭代预算和取消信号。 |
| `loop.py` | 执行 Agent 主循环：调用 LLM、调度工具、保存消息并处理压缩。 |
| `prompt.py` | 构建系统提示词，汇总身份、配置、记忆、技能和上下文文件。 |

## 注意事项

- 修改 `loop.py` 时要同步关注工具注册、会话存储和上下文压缩的调用契约。
- 修改 `prompt.py` 时要避免把运行时召回内容和用户当前输入混在同一语义边界里。
